# LPS / LCov / Omics Program

This directory holds the active cross-project program roadmap that connects
`geosmooth` method development to downstream biological applications.

The program spans:

- local polynomial smoothers, including LPS and PS-LPS;
- local covariance / association models;
- compositional synthetic-data models;
- temporal 16S rRNA community-type models;
- disease, adverse-outcome, and multi-omics association models;
- metagenomic co-abundance gene cluster applications.

Some downstream application work does not technically belong inside the
`geosmooth` package. It is still included in this roadmap because the
development of LPS-type smoothers and local covariance models is driven by
those biological use cases. Downstream implementation artifacts should live in
their own project repositories and be linked from this program plan.
