K4 LPS Local-PCA Native Prototype

This benchmark compares the existing R local-PCA path in fit.lps() with the explicit native prototype backend = "cpp.local.pca". The package default backend = "auto" remains unchanged.

The validation criterion is numerical agreement of candidate CV RMSE and final fitted values, plus a simple elapsed-time comparison on existing K3.8 VALENCIA-derived embedding assets.
