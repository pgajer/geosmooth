# Geosmooth Project Briefs

This directory contains timestamped, agent-facing project briefs for durable
context transfer. These briefs are different from phase handoffs, dashboards,
and progress reports: each one is meant to orient a new agent or auditor to the
current state of a substantial project thread.

The Markdown file is canonical. Matching HTML files are rendered from the
Markdown for human browsing.
